PyFit is a collection of exercise calculations

PyFit structures exercise physiology equations to allow you to focus more on the results of the calculations, rather than performing the calculations themselves. Many exercise physiology equations were calculated using different units of measure. PyFit is a compilation of these researcher's efforts (sources included) for developers to utilize.

Quick Start

The library is mainly used with the Person class. The Person class is composed of methods and instances of other classes you can use to make any necessary calculations.

Person has many subclasses for different types of people. The subclasses have different methods/implementations to account for the difference in equations. This is to ensure the methods only return results relevant to that Person instance Rather than trying to determine yourself which subclass of Person you need, it is easiest to use the Person_Factory to return an instance of the right subclass.

Installation

  1. Download the PyFitness Library

  2. Setup the library

    
 // create an instance of the Person_Factory
 johndoe@john-desktop:~$ python /path/to/PyFit/setup.py install

  3. Get Started!


# create an instance of the Person_Factory
factory = Person_Factory();
  
# create an instance of datetime as my date of birth
dob = datetime.datetime(1990,3,17);
 
# create a Person instance or subclass of Person
me = factory.create("John Doe", "male", dob, 54, 154)
	
# execute a method from Person
me.predicted_tee()['active'];  # 2537.0212935729905 (kcal)
	
# or from one of its components
me.cardio.heart_rate_max();  # 190.538 (bpm)
